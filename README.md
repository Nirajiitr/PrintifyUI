
## installatins 

To get started with the project, you need to have Node.js and npm installed. Clone the repository and install the dependencies:

```bash
    git clone https://github.com/Nirajiitr/PrintifyUI.git
    cd PrintifyUI/printify
    npm install

```
### Run 
     npm run div
